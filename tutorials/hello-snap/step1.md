Welcome to **Hello Snap** — a hands-on tutorial that takes you from a plain C program all the way to a fully confined [snap](https://snapcraft.io/) package ready to run on **Ubuntu Core**.

## What you will learn

- How to compile a C application on Ubuntu Server
- How snaps provide application confinement
- How to write a `snapcraft.yaml` descriptor
- How to build a snap with **Snapcraft**
- How to sideload and test a snap

## What is a snap?

A snap is a self-contained application bundle for Linux. It includes the app and all its dependencies, runs in a sandbox, and can be installed on any snap-enabled system — including **Ubuntu Core**, a minimal OS designed for IoT and embedded devices.

## The environment

Your terminal on the right is a live **Ubuntu 24.04** container managed by LXD.  
You will do all your development work there.

- Commands highlighted with a **▶ Run** button can be executed with a single click.
- You can also type directly in the terminal at any time.

When you are ready, click **Next →** to begin setting up your build tools.
