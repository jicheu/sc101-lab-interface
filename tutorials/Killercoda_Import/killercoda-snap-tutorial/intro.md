# Understanding what confinement means

Welcome to this interactive tutorial! You will learn how to:
1. Write a C++ application that fetches a random inspirational message using an API and writes it to a file.
2. Package the application as an unconfined snap (`devmode`) to test it.
3. Enforce strict confinement and resolve permission issues by connecting interfaces.
4. Run the strictly confined snap inside an emulated Ubuntu Core image using `qemu`.

Let's get started!
