Follow guidance for directions.md

Name this training "Understanding what confinement means"

## Step 1
Create a cpp application that will ask for a filename, and write such file with a random inspirational message

Provide this inspirational message from a website which provides a free api to get a different message everytime.

## Step 2
Create a snapcraft.yaml file to run the app as a snap on Ubuntu Desktop, unconfined

## Step 3
Change the snap to strictly confined. Let the user run into the issue that they need file access, then correct it and offer to run the snap on a core image emulated on lxd
